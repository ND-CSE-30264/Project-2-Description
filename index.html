CSE 30264 Computer Networks at ND - Spring 2024

Semester Project 2: Analysis of Network Traffic Traces

Overview

In this project, you will need to analyze network traffic traces collected from different applications.

The objective of this project is to understand how real network traffic is structured, understand how to analyze network traffic, as well as understand its characteristics and properties.

You are encouraged to use Wireshark as well as Python and other programming languages to inspect and analyze the provided network traffic traces.

Network Traffic Traces

You are given four network traffic traces (trace 1, trace 2, trace 3, and trace 4). Each trace contains the network traffic generated by a different application. Applications are kept secret for the time being. It will be part of your project to look through the traces and find clues that will help you identify each application. All traces contain information about all layers of the TCP/IP protocol stack (link layer all the way up to the application layer). You can find the traffic traces here.

Trace 1

Use Wireshark to open trace 1. This trace contains about 30 seconds of traffic. Feel free to inspect traffic through Wireshark or export the traffic as .csv files (or files of other formats) and use scripts to analyze it. For this trace, let's think through and answer the questions below.

Questions

  • Q1: What is the application that generated the traffic in this trace? How did you come up with this answer? (2.5 pts)

  • Q2: What are the two IP addresses used in the context of this application? Are they public or private IP addresses? (2.5 pts)

  • Q3: Let's call the two IP addresses used IP1 and IP2. How many packets and bytes of traffic were sent from IP1 to IP2 and from IP2 back to IP1? How did you come up with your answer? (2.5 pts)

  • Q4: Are there any losses experienced during the operation of this application? If so, how many and why they might have happened? (2.5 pts)

  • Q5: Considering the answer that you gave for Q1, what information was exchanged between the two IP addresses? (5 pts)
